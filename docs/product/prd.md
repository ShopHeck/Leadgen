# Product: CloserFlow AI

## One-liner
AI-powered platform that captures, qualifies, nurtures, and books high-ticket leads automatically.

## Target Users
- Med spas
- Roofers
- Solar companies
- Law firms
- Agencies

## Core Problem
Businesses generate leads but:
- slow response time
- poor qualification
- low booking rates
- no follow-up system

## Solution
Automated system:
Traffic → Funnel → Qualification → AI Follow-Up → Booking → CRM → Revenue Tracking

## Core Features
- Funnel builder (MVP = templates)
- Lead capture + multi-step forms
- AI qualification + scoring
- CRM pipeline
- SMS/email automation
- Calendar booking
- Attribution dashboard

## Success Metrics
- Lead → booking %
- Response time < 60 seconds
- Show rate
- Close rate
- Revenue per lead

## Monetization
- $297 / $497 / $997 SaaS tiers
- usage-based SMS + AI credits
- optional setup fees

## MVP Scope
IN:
- auth + workspace
- 1 funnel template
- form submission
- lead scoring
- CRM
- SMS/email automation
- booking integration
- dashboard
- Stripe billing

OUT:
- drag/drop builder
- voice AI
- marketplace
